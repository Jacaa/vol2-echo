## How to start
1. Clone:
```bash
$ git clone https://github.com/netguru-code-college/vol2-echo.git
```

2. Enter directory:
```bash
$ cd vol2-echo
```

3. Instal gems:
```bash
$ bundle
```

4. Create and seed database:
```bash
$ rails db:create
$ rails db:seed
```

5. Run rails server:
```bash
$ rails s
```

Enjoy!
